# 接口名称
主牙列检测
# 接口描述
该接口用于口内照片的牙列检测。
# 请求URL
`POST /engine/v2/inner/main-jaw`
# 请求头
需要在请求头中加入鉴权信息
# 回调地址
# 输入定义


| 参数          | 类型     | 说明                                |
| ----------- | ------ | --------------------------------- |
| sid         | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| rawData.url | String | 口内照片的URL地址，请保证可用                  |
| callback    | Object | [回调](/入门/回调)                      |

# 输入示例
```json
{  
    "sid": "{{$guid}}",  
    "rawData": {  
        "url": "https://XXXXXXX.jpg"  
    },  
    "callback": [  
        {  
            "type": "rabbitmq",  
            "out": "aaaaa"  
        }  
    ]  
}
````


# 输出定义
| 参数                     | 类型     | 说明                                |
| ---------------------- | ------ | --------------------------------- |
| sid                    | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| code                   | Int    | 接口返回code码，HTTP状态码                 |
| errMsg                 | String | 接口错误信息                            |
| result                 | Object | 返回的详细数据对象                         |
| result.left_xywh_list  | Arry   | 左侧牙列分割结果                          |
| result.right_xywh_list | Arry   | 右侧牙列分割结果                          |
| result.is_rotate       | bool   | 是否上下翻转                            |
