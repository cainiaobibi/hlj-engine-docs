# 接口名称
口扫分析
接口编码：oral-scan-v2
# 接口描述
本接口对患者的口内扫描（Intraoral Scan）上下颌3D模型进行AI分析，自动识别关键解剖标志点，并计算出弓形宽度、牙齿位置、覆合覆盖关系等相关的数字化测量数据（KPI）
**使用场景建议：**

- **正畸辅助诊断与方案设计：** 快速获取牙弓、牙齿排列、咬合关系的精确数据，为医生制定个性化矫正方案提供量化依据。
- **生产诊断报告：** 自动生成包含关键测量指标（如牙弓宽度、覆合覆盖、Spee曲线等）的标准化诊断报告，简化病历记录与存档流程。
- **治疗过程监控与效果评估：** 通过对比不同治疗阶段的口扫数据，量化评估牙齿移动情况与治疗进展，方便医患沟通。

**注意事项：**

- 请确保`lowerUrl`和`upperUrl`为可公开访问的有效链接。
- 上传的模型文件必须为STL格式。
- 为保证分析准确性，请确保口扫模型数据完整、无明显变形或缺失。
- 请求中传入的`sid`需保证唯一，以便于后续的任务追踪与结果查询。
# 请求URL
`POST https://api-dev.cloud.hailijia.cn/engine/v2/mesh/oral-scan-v2`
