# 接口名称
全景片
# 接口描述
该接口用于上传一张全景片，将输入全景片进行AI分析，输出关键点位置和KPI结果。
# 请求URL
`POST /engine/v2/tomography/tomography`
# 请求头
需要在请求头中加入鉴权信息
# 回调地址
# 输入定义


| 参数                    | 类型     | 说明                                |
| --------------------- | ------ | --------------------------------- |
| sid                   | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| rawData.tomographyUrl | String | 全景片图像的公网地址请保证链接可用                 |
| ct_json_data          | Object | 侧位片的请求示例和返回值组成的一个对象               |
| callback              | Object | [回调](/入门/回调)                      |

# 输出定义
| 参数                                     | 类型     | 说明                                |
| -------------------------------------- | ------ | --------------------------------- |
| sid                                    | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| code                                   | Int    | 接口返回code码，HTTP状态码                 |
| errMsg                                 | String | 接口错误信息                            |
| result                                 | Object | 返回的详细数据对象                         |
| result.rawData.url                     | Object | 图片远程地址                            |
| result.kpi                             | Object | 根据点结合医学计算出来的指标                    |
| result.landmark                        | Object | 计算出来的点坐标                          |
| result.metadata.{fid}.contour          | Arry   | 实力分割边缘点                           |
| result.metadata.{fid}.merge            | bool   | 是否萌生牙                             |
| result.metadata.{fid}.num              | String | 牙齿编号                              |
| result.metadata.{fid}.quadrant         | String | 在全景片中医学标准的牙齿4个分区quadrant表示所在区域    |
| result.metadata.{fid}.width_dist       | Float  | 牙齿基于当前像素宽度                        |
| result.metadata.{fid}.first_point      | Arry   | 进中点位置                             |
| result.metadata.{fid}.second_point     | Arry   | 远中点位置                             |
| result.metadata.{fid}.crown_root_point | Arry   | 牙冠点位置                             |
| result.metadata.{fid}.width            | Float  | 牙齿基于实际物理宽度                        |
