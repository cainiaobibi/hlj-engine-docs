# 回调
回调说明：

| 参数               | 类型     | 说明                                                                                                   |
| ---------------- | ------ | ---------------------------------------------------------------------------------------------------- |
| callback[0].type | String | 异步队列的队列类型                                                                                            |
| callback[0].out  | String | 异步队列的队列名称,请求的队列名称，如果在拥有默认队列的情况下额外开启一个监听，当开启的这个额外的监听的队列名称和请求时传入的监听名称一致那么将会被这个额外的监听到，如果不一致那么就会进入到默认的监听 |
```json
{  
  "callback": [  
    {  
      "type": "rabbitmq",  
      "out": "aaaaa"  
    }  
  ]  
}
```
