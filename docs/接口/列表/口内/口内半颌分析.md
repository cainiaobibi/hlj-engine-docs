# 接口名称
口内半颌分析
# 接口描述
该接口用于半颌的口内照片进行计算分析。
# 请求URL
`POST /engine/v2/inner/half`
# 请求头
需要在请求头中加入鉴权信息
# 回调地址
# 输入定义


| 参数                                     | 类型     | 说明                                                                 |
| -------------------------------------- | ------ | ------------------------------------------------------------------ |
| sid                                    | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性                                  |
| rawData.url                            | String | 口内半颌的URL地址，请保证可用                                                   |
| rawData.half                           | String | 口内半颌的类型，是固定值“upper”,"lower"                                        |
| rawData.main_area_info.left_xywh_list  | Arry   | 左侧牙列分割数据（来源于主牙列检测）[来源](/接口/列表/口内/主牙列检测?id=输出示例)                    |
| rawData.main_area_info.right_xywh_list | Arry   | 右侧侧牙列分割数据（来源于主牙列检测）[来源](/接口/列表/口内/主牙列检测?id=输出示例)                   |
| rawData.main_area_info.is_rotate       | bool   | 图片石头选择120°固定值“true”,"false"（来源于主牙列检测）[来源](/接口/列表/口内/主牙列检测?id=输出示例) |
| callback                               | Object | [回调](/入门/回调)                                                       |
