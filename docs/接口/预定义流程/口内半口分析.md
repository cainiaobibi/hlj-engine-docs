# 接口名称
口内半口分析
# 接口描述
该接口用于口内半口分析。
# 请求URL
`POST /engine/v2/inner/inner-half-all`
# 请求头
需要在请求头中加入鉴权信息
# 回调地址
# 输入定义


| 参数           | 类型     | 说明                                |
| ------------ | ------ | --------------------------------- |
| sid          | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| rawData.url  | String | 口内照片的URL地址，请保证可用                  |
| rawData.half | String | upper / lower 口内照上颌或下颌            |
| callback     | Object | [回调](/入门/回调)                      |
# 输入示例
```json
{
  "sid": "{{$guid}}",
  "rawData": {
    "url": "https://XXXXXXX.jpg",
    "half": "upper"
  },
  "callback": [
    {
      "type": "rabbitmq",
      "out": "aaaaa"
    }
  ]
}
````


# 输出定义
| 参数                                           | 类型     | 说明                                |
| -------------------------------------------- | ------ | --------------------------------- |
| sid                                          | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| code                                         | Int    | 接口返回code码，HTTP状态码                 |
| errMsg                                       | String | 接口错误信息                            |
| result                                       | Object | 返回的详细数据对象                         |
| details                                      | Object | 包含牙齿关键点信息对象                       |
| result.details.11                            | Object | 单个牙齿的相关信息                         |
| result.details.11.mesial_px                  | Arry   | 近中点坐标                             |
| result.details.11.distal_px                  | Arry   | 远中点坐标                             |
| result.details.11.score                      | Float  | 置信度                               |
| result.details.11.size_px                    | Float  | 像素宽度                              |
| result.details.11.size_mm                    | Float  | 实际宽度                              |
| result.details.11.mm_per_px                  | Float  | 单位像素宽度                            |
| result.details.kpi.inner_{fid}_{fid}         | Float  | 相邻牙近远中点欧氏距离                       |
| result.details.kpi.inner_{fid}_{fid}-project | Float  | 相邻牙间隙                             |
| result.details.kpi.inner_crown_length        | Float  | 牙冠长度和                             |
| result.details.kpi.inner_arc_length          | Float  | 牙弓长度                              |
| result.details.kpi.inner_crowding_length     | Float  | 拥挤度                               |
| result.details.kpi.inner_uw0                 | Float  | 牙弓前段宽（3-3）                        |
| result.details.kpi.inner_uw1                 | Float  | 牙弓中段宽（4-4）                        |
| result.details.kpi.inner_uw2                 | Float  | 牙弓后段宽（6-6）                        |
