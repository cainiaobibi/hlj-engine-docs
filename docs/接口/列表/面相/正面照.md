# 面像正面照分析

# 接口描述：
该接口用于上传一张面部正面照片，将输入正面照进行AI分析。

# 请求URL
`POST /engine/v2/face/face-front`

# 请求头:
需要在请求头中加入鉴权信息

# 回调地址：

# 输入定义：

| 参数          | 类型     | 说明                                                                                        |
| ----------- | ------ | ----------------------------------------------------------------------------------------- |
| sid         | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性                                                         |
| rawData.url | String | 图像的公网地址请保证链接可用                                                                            |
| callback    | Object | [回调](obsidian://open?vault=Obsidian%20Vault&file=%E5%85%A5%E9%97%A8%2F%E5%9B%9E%E8%B0%83) |

# 输入示例：
```json
{  
  "sid": "{{$guid}}",  
  "rawData": {  
    "url": "https://XXXXXX.jpg"  
  },  
  "callback": [  
    {  
      "type": "rabbitmq",  
      "out": "aaaaa"  
    }  
  ]  
}
````
