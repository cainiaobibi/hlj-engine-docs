# 接口名称
头影重叠
# 接口描述
该接口用于两张侧位片，或一张侧位片 一张侧面照，将输入的图片进行重叠配准，输出关键点位置。
# 请求URL
`POST /engine/v2/ceph/ct-overlap`
# 请求头
需要在请求头中加入鉴权信息
# 回调地址
# 输入定义


| 参数          | 类型     | 说明                                |
| ----------- | ------ | --------------------------------- |
| sid         | String | 唯一ID可以传入任意String，用于回调查询等操作，请确保唯一性 |
| rawData.src | Object | 待配准图像的点数据                         |
| rawData.dst | Object | 目标图像的点数据                          |
| callback    | Object | [回调](/入门/回调)                      |
